## üìñ Introduction

In the journey of language learning, context is king. Traditional vocabulary apps often present words in isolation, leading to rote memorization that fades quickly. **LingoPick** was born from a simple product philosophy: **capture the moment of curiosity**.

LingoPick is a browser extension that allows users to instantly translate and save words *within the context* of the article they are reading. By leveraging advanced AI, it provides context-aware definitions and seamlessly integrates into the user's browsing flow. It bridges the gap between passive reading and active vocabulary acquisition.

### Key Value Proposition
- **Contextual Learning:** Save words with their original sentence to retain usage context.
- **Seamless UX:** Non-intrusive popup interactions that don't break the reading flow.
- **AI-Powered:** Utilizes **Google Gemini** and **DeepSeek** to generate precise, high-quality definitions and collocations.

## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/lingopick.git
   cd lingopick
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file or configure WXT secrets with your API keys:
   - `WXT_SUPABASE_URL` & `WXT_SUPABASE_KEY`
   - `WXT_GEMINI_API_KEY`
   - `WXT_DEEPSEEK_API_KEY` (optional)
   - `WXT_GUMROAD_PRODUCT_ID`

4. **Run Development Server:**
   ```bash
   # For Chromium (Chrome, Edge, Brave)
   npm run dev

   # For Firefox
   npm run dev:firefox
   ```

5. **Load Extension:**
   - **Chrome:** Go to `chrome://extensions`, enable "Developer mode", and load the unpacked extension from the `.output/chrome-mv3` folder.

## üí° Highlights & Engineering Thoughts

### 1. Architecture: Service-Oriented Logic
To ensure the codebase remains clean and scalable, I separated business logic from UI components. The `libs/` directory contains specialized services:
- **`DatabaseService`**: Abstracts all Supabase interactions.
- **`TranslationService`**: Handles the switching logic between different AI providers (Gemini/DeepSeek) and prompt engineering.
- **`LicenceValidationService`**: Encapsulates the Gumroad verification logic.

### 2. AI Integration Strategy
Instead of relying on rigid dictionary APIs, LingoPick uses LLMs to generate definitions. This allows for:
- **Context Awareness:** The prompt includes the surrounding sentence, ensuring the definition matches the specific usage (e.g., "bank" as a river edge vs. financial institution).
- **Flexibility:** The `Prompts` class manages structured JSON outputs from the LLMs, making the UI rendering deterministic despite the AI's creative nature.

### 3. Modern React & Performance
- Adopted **React 19** to leverage the latest rendering improvements.
- Used **TailwindCSS v4** for a zero-runtime overhead styling solution.
- Implemented **LocalForage** to cache user settings and minimize unnecessary network requests.

### 4. Future Roadmap
- **Spaced Repetition System (SRS):** Implement an algorithm (like Sm-2) for the flashcard feature.
- **Multi-language Support:** Expand beyond English-to-Target translation.
- **Cross-Platform:** Explore migrating the core logic to React Native for a mobile companion app.
